# Java运算符优先级

| **运算符**                        | **结合性** |
| :-------------------------------- | :--------- |
| [ ] . ( ) (方法调用)              | 从左向右   |
| ! ~ ++ -- +(一元运算) -(一元运算) | 从右向左   |
| * / %                             | 从左向右   |
| + -                               | 从左向右   |
| << >> >>>                         | 从左向右   |
| < <= > >= instanceof              | 从左向右   |
| == !=                             | 从左向右   |
| &                                 | 从左向右   |
| ^                                 | 从左向右   |
| \|                                | 从左向右   |
| &&                                | 从左向右   |
| \|\|                              | 从左向右   |
| ?:                                | 从右向左   |
| =                                 | 从右向左   |
